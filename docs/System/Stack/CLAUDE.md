## Tech Stack Overview

**Frontend**: Vue 3 (Composition API), Vuetify 3, Tailwind CSS
**Build**: Vite
**State**: Pinia
**Routing**: Vue Router 4 (hash-based)
**Backend**: Firebase (Auth, Firestore, Storage)
**Testing**: Vitest
